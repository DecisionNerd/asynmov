## What is Asynmov?

Asynmov is a tool for generating large corpora of synthetic data for world building. It exposes a Python API backed by a high-performance Rust core, making it capable of producing the scale of data that deep world building requires — thousands of characters, events, relationships, and timelines — without sacrificing speed.
